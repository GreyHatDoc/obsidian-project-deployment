# Project Deployment

# Goal
- To simplify the deployment of both plugins and templates into a project directory so that we can get up and running quickly

# Use Cases

## New vault
    - Need to build out entire oblivian structure with .oblivian folder (will use .oblivion folder in current repo)
    - Need to then add plugins and turn on core plugins such as templates
    - Need to pull in templates and deploy project templates
    - Need to pull in plugins and add to json for easy enablement
    - Need to create or update gitignore to add .oblivian folder
## Existing Vault
    - Need to pull in plugins and deploy in folder
    - Need to pull in templates and deploy project templates
    - Need to add to or create community plugin json
  
Need to deploy in powershell and bash (will need to update template deployment for powershell also)
